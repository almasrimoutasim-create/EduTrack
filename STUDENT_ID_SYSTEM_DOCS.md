# نظام أرقام الطلاب المدرسية - نموذج شامل

## 📋 نظرة عامة
تم بناء نظام متكامل وتلقائي لإنشاء وإدارة وعرض أرقام الطلاب المدرسية بصيغة موحدة (0001, 0002, إلخ).

## ✨ المميزات الرئيسية

### 1️⃣ **الإنشاء التلقائي للرقم المدرسي**
- عند إضافة طالب جديد، يتم إنشاء الرقم تلقائياً
- الصيغة: 4 أرقام مع أصفار بادية (0001, 0002, 0003, etc.)
- يتم البحث عن أعلى رقم موجود وإضافة 1

**الملفات المسؤولة:**
- `src/components/students/StudentForm.jsx` - نموذج إنشاء الطالب
- `src/utils/studentIdFormatter.js` - دوال التنسيق

### 2️⃣ **العرض الاحترافي للرقم**
- عرض بارز في صفحة الطالب الرئيسية
- مكون متخصص `StudentIDCard` بتصاميم مختلفة
- نسخ الرقم بضغطة زر واحدة

**الملفات المسؤولة:**
- `src/components/student-dashboard/StudentIDCard.jsx` - مكون العرض
- `src/components/student-dashboard/StudentProfileCard.jsx` - بطاقة الملف الشخصي
- `src/pages/StudentPortal.jsx` - صفحة الطالب الرئيسية

## 🔧 دوال المساعدة

### `formatStudentId(studentId)`
تنسيق رقم الطالب بصيغة موحدة
```javascript
formatStudentId("1")     // → "0001"
formatStudentId("42")    // → "0042"
formatStudentId("STU-5") // → "0005"
```

### `getNextStudentId(students)`
حساب الرقم التالي بناءً على الطلاب الموجودين
```javascript
const students = [
  { student_id: "0001" },
  { student_id: "0002" }
];
getNextStudentId(students) // → "0003"
```

### `isValidStudentId(studentId)`
التحقق من صحة صيغة الرقم المدرسي
```javascript
isValidStudentId("0001")    // → true
isValidStudentId("12345")   // → false (أكثر من 4 أرقام)
isValidStudentId("")        // → false
```

### `displayStudentId(studentId)`
تحويل الرقم للصيغة المعروضة
```javascript
displayStudentId("42")   // → "0042"
displayStudentId("0001") // → "0001"
```

## 📊 تدفق النظام

### عند إنشاء طالب جديد:
```
1. يفتح نموذج StudentForm
2. يجلب جميع الطلاب الحاليين (useQuery)
3. يحسب الرقم التالي باستخدام getNextStudentId()
4. يعرض الرقم في الحقل (معطّل من التعديل - auto-generated)
5. عند الحفظ، يُحفظ الرقم مع البيانات الأخرى
```

### عند عرض بيانات الطالب:
```
1. يتم جلب بيانات الطالب من قاعدة البيانات
2. يتم تنسيق الرقم باستخدام formatStudentId()
3. يُعرض في:
   - StudentProfileCard (بطاقة الملف الشخصي)
   - StudentIDCard (مكون مخصص بتصاميم مختلفة)
   - StudentPortal (صفحة الطالب الرئيسية)
```

## 🎨 مكونات العرض

### StudentIDCard
عرض احترافي للرقم المدرسي مع ميزات إضافية:
- **حجم قابل للتخصيص**: `sm`, `md`, `lg`
- **نسخ سريعة**: انسخ الرقم بضغطة زر
- **عرض الاسم**: يمكن عرض اسم الطالب معه
- **تصميم جميل**: تدرج لوني وأيقونات

```javascript
<StudentIDCard 
  studentId="42" 
  studentName="أحمد محمد"
  size="md"
/>
```

### StudentProfileCard
بطاقة الملف الشخصي التي تعرض:
- صورة الطالب
- الاسم والحالة
- **الرقم المدرسي بارزاً** ✨
- الصف والفصل
- معلومات الحافلة

## 📱 الاستخدام

### إضافة طالب جديد:
1. انتقل إلى صفحة إدارة الطلاب
2. انقر "إضافة طالب جديد"
3. ملء البيانات (الاسم إلزامي)
4. سيتم إنشاء الرقم تلقائياً (0001, 0002, etc.)
5. انقر "حفظ"

### عرض رقم الطالب:
- **في الصفحة الرئيسية**: يظهر الرقم بارزاً مع أيقونة ID
- **في بطاقة الملف**: معروض في صندوق بتصميم جميل
- **في جميع الصفحات**: يمكن نسخ الرقم بضغطة زر
