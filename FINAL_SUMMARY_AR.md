## 🚀 كيفية الاستخدام

### 1️⃣ إضافة طالب جديد
```
1. افتح صفحة إدارة الطلاب
2. انقر "إضافة طالب جديد"
3. املأ البيانات (الاسم والبيانات الأخرى)
4. الرقم ينشأ تلقائياً (0001, 0002, إلخ)
5. انقر "حفظ"
```

### 2️⃣ عرض الرقم في المكونات
```javascript
import StudentIDCard from '@/components/student-dashboard/StudentIDCard';

<StudentIDCard 
  studentId="42" 
  studentName="أحمد محمد"
  size="md"
/>
```

### 3️⃣ استخدام الدوال
```javascript
import { formatStudentId, getNextStudentId } from '@/utils/studentIdFormatter';

const formatted = formatStudentId("42");      // "0042"
const nextId = getNextStudentId(allStudents); // "0005"
```
